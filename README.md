# (C++) SFML Network Ping Pong
A C++17 network Ping Pong game develop with SFML.

## Education Project
This is part of the **AI Team** project for agentic workflow. In this project the AI will be like a copilot and assist the developer.


## Development Environment
```
# Windows (MSYS2)
pacman -S \
    mingw-w64-ucrt-x86_64-gcc \
    mingw-w64-ucrt-x86_64-cmake \
    mingw-w64-ucrt-x86_64-ninja \
    mingw-w64-ucrt-x86_64-gtest \
    mingw-w64-ucrt-x86_64-sfml

# Unix (Ubuntu)
sudo apt install cmake gcc libsfml-dev libgtest-dev
```
